  • Swaggerhub
  • Resources API definition

Endpoints

The following endpoints can be reached:

  • GET {url}/resources/ids
    To get all resource ids. Command: 
    curl -X GET "{url}/resources/ids" -H  "accept: application/vnd.api+json"
  • POST {url}/resources
    To add a new resource. Command: 
    curl -X POST "{url}/resources" -H  "accept: application/vnd.api+json" -H  "content-type: application/vnd.api+json" -d "{{  \\"data\\": {{    \\"type\\": \\"resource\\",    \\"attributes\\": {{ 
\\"title\\": \\"Example Website\\",      \\"url\\": \\"https://example.org\\",      \\"licenses\\": [],      \\"mimeType\\": \\"text/html\\",      \\"contentCategory\\": \\"l\\", 
         \\"languages\\": [        \\"en-en\\"      ],      \\"thumbnail\\": \\"http://cache.schul-cloud.org/thumbs/k32164876328764872384.jpg\\"    }},    \\"id\\": \\"cornelsen-physics-1\\"  }}}}"
  • DELETE {url}/resources
    To remove all saved resources. Command: 
    curl -X DELETE "{url}/resources" -H  "accept: application/vnd.api+json"
  • GET {url}/resources/{{resourceId}}
    To get a specific resource. Command: 
    curl -X GET "{url}/resources/cornelsen-physics-1" -H  "accept: application/vnd.api+json"
  • DELETE {url}/resources/{{resourceId}}
    To delete a specific resource. Command: 
    curl -X DELETE "{url}/resources/cornelsen-physics-1" -H  "accept: application/vnd.api+json"
